* Description de l'application :

 * Namer App (Flutter)

Namer App est une application Flutter simple qui génère des paires de mots aléatoires et permet à l’utilisateur de les ajouter à une liste de favoris.
L’application utilise une architecture moderne avec Provider pour la gestion d’état et s’adapte aux différentes tailles d’écran.

   * Fonctionnalités

Génération aléatoire de paires de mots

Ajout / suppression des mots dans les favoris

Navigation entre les pages Home et Favorites

Interface responsive avec NavigationRail

Design moderne basé sur Material 3

   * Technologies utilisées

Flutter

Dart

Provider (gestion d’état)

Material 3

  *  Objectif du projet

Ce projet a pour objectif de :

Apprendre les bases de Flutter

Comprendre la gestion d’état avec ChangeNotifier & Provider

Mettre en place une navigation simple

Manipuler des listes dynamiques et des widgets Flutter
